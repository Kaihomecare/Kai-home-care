# Kai Home Care

Marketing site for Kai Home Care, in-home care serving Oʻahu.

Static site hosted on Netlify. Plain HTML/CSS, no build step.

## Pages
- `index.html` — landing page
- `care-estimate.html` — care calculator (lead capture)
- `careers.html` — job interest form + resume upload
- `contact.html` — contact info

## Editing
Edit files directly in GitHub. Netlify auto-deploys on commit.
